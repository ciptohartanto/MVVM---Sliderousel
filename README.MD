# Getting Started

### Step 1 - Clone

```bash
git clone https://github.com/ciptohartanto/MVVM---Sliderousel.git
```

### Step 2 -Installation
```bash
nmp install
```

### Step 3 - Go live

To live preview:
```bash
npm run dev
```

To generate `dist/`
```bash
npm run dist
```
